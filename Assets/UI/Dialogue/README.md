# Dialogue UI

Runtime loads `Resources/Dialogue/DialoguePanel.prefab` (built hierarchy + registry).

For a fully wired prefab with serialized layout references, run **Tools → Dialogue → Setup Dialogue UI** in the Unity Editor. That regenerates this folder’s prefab, `ChoiceRow.prefab`, and character assets under `Assets/Dialogue/Characters/`.

- Joe: `joe_schmo_fosho.png` → **Joe Schmo FoSho**
- Donald: `donald_musk.png`
- Lily Chen (dialogue): no portrait until art is added to `Lily.asset`
- Lily Pad (FaceWeb tab): no portrait yet — add `lily_pad.png` and assign on `LilyPad.asset` when ready
